Functions in JavaScript

To understand this article you need to know:

  • Fundamentals of JavaScript (variables, operator, etc)
  • Loops and fork (if, for, while, etc)
  • Fundamentals of HTML (and how to work with HTML and JavaScript together at browser)

What are functions:

Functions are a piece of code that not follow the regular flow of the program (top to bottom). When the compiler sees a function, all code inside it is ignored and the program starts the reading again right after the end of the function.

But when you want this code runs, you have to call it(or invoke, it is the same thing). Then the program enters the function and executes like a regular code, and after its ends, the program comes back to the line of the calling.

Practice time:

In this exercise, my objective will be to dynamically change the text in HTML, I strongly recommend that you write the code in our own text editor and test each step of the process.

  • First lets write some inicial code to work:
    • <body>
    <h1 id="my-text">My first text</h1>
    <script>
    let text = document.getElementById("my-text");
    </script>
    </body>
  • Now if I want to change this text, I can write just one more line of code
    <body>
    <h1 id="my-text">My first text</h1>
    <script>
    let text = document.getElementById("my-text");
    text.innerHTML = "Modified Text";
    </script>
    </body>

  • Let's do the same thing using a function?

    First I will wrap the "modified text" command in a function

    <body>
    <h1 id="my-text">My first text</h1>
    <script>
    let text = document.getElementById("my-text");
    function changeText(){
    text.innerHTML = "Modified Text";
    }
    </script>
    </body>

  • Did you test? What happened?

    Exactly, nothing happened... do you know why? Click here to go back to the explanation of the function and try to figure out

    As you may notice, I write the function but I never call her, so let's do this right now:

    <body>
    <h1 id="my-text">My first text</h1>
    <script>
    let text = document.getElementById("my-text");
    function changeText(){
    text.innerHTML = "Modified Text";
    }
    changeText()
    </script>
    </body>

  • Yes, it is that simple... to call a function you have to write its name and one pair of parentheses "()" right after. Now everything is work again.

    Theses parenthesis, also serve to enter with some data to use inside the function. It is called "arguments".

    Let's pass the new text of the element as an argument.

    <body>
    <h1 id="my-text">My first text</h1>
    <script>
    let text = document.getElementById("my-text");
    function changeText(newText){
    text.innerHTML = newText;
    }
    changeText("My new text")
    </script>
    </body>

  • Pretty awesome, isn't it? Now any text inside the parentheses when the function is called is inserting in the place.

    And even the element itself can be a argument.

    <body>
    <h1 id="my-text">My first text</h1>
    <script>
    let text = document.getElementById("my-text");
    function changeText(element, newText){
    text.innerHTML = newText;
    }
    changeText(text, "My new text")
    </script>
    </body>

Why you should use Functions

In this particular case, maybe it looks like the code became more complex after the function, so why use it?

  • Better than explain, let me show to you:

    Now you understand how functions works, let's increase the complexity of the code and do the same thing we did before But this time there are two elements, and we will change two properties in each

    <body>
    <h1 id="my-text">My first text</h1>
    <h1 id="my-other-text">My other text</h1>
    <script>
    let text = document.getElementById("my-text");
    let otherText = document.getElementById("my-other-text");
    text.innerHTML = "Modified Text";
    text.style.color = "blue";
    otherText.innerHTML = "Modified Second Text";
    otherText.style.color = "blue";
    </script>
    </body>
  • Using function with arguments
    • <body>
    <h1 id="my-text">My first text</h1>
    <h1 id="my-other-text">My other text</h1>
    <script>
    let text = document.getElementById("my-text");
    let otherText = document.getElementById("my-other-text");
    function changeTextAndColor(element, newText, color){
    element.innerHTML = newText;
    element.style.color = color;
    }
    changeTextAndColor(text, "First: Blue Text", "blue");
    changeTextAndColor(otherText, "Second: Red Text", "red");
    </script>
    </body>
  • Which code do you think is easier to read and understand... and change? And if it were more actions to do or more elements to contemplate, which code is became bigger?

Conclusion

Using functions help to maintain to code organized and clean. Besides helping to segment the code, so who is reading it and trying to understanding (other programmers and even you in the future) don't have to read all the code once. The code now can by readed by "chapters".

This is only an introduction to the theme, functions are a key piece in JavaScript, we will dig more into this in future articles.
